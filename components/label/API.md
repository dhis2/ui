### Label

#### Usage

**Note**: If possible, import the component from the main UI (`@dhis2/ui`) package.

```js
import { Label } from '@dhis2-ui/label'
```

#### Props

| Name      | Type    | Default                | Required | Description |
| --------- | ------- | ---------------------- | -------- | ----------- |
| children  | string  |                        |          |             |
| className | string  |                        |          |             |
| dataTest  | string  | `'dhis2-uicore-label'` |          |             |
| disabled  | boolean |                        |          |             |
| htmlFor   | string  |                        |          |             |
| required  | boolean |                        |          |             |
