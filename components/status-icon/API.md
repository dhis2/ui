### StatusIcon

#### Usage

**Note**: If possible, import the component from the main UI (`@dhis2/ui`) package.

```js
import { StatusIcon } from '@dhis2-ui/status-icon'
```

| valid   | boolean |         |          |             |
| warning | boolean |         |          |             |
