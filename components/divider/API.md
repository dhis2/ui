### Divider

#### Usage

To use `Divider`, you can import the component from the `@dhis2/ui` library  


```js
import { Divider } from '@dhis2/ui'
```


#### Props

|Name|Type|Default|Required|Description|
|---|---|---|---|---|
|className|string||||
|dataTest|string|`'dhis2-uicore-divider'`|||
|dense|boolean||||
|margin|string|``${spacers.dp8} 0``|||
