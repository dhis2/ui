### Help

#### Usage

**Note**: If possible, import the component from the main UI (`@dhis2/ui`) package.

```js
import { Help } from '@dhis2-ui/help'
```

#### Props

| Name      | Type   | Default               | Required | Description |
| --------- | ------ | --------------------- | -------- | ----------- |
| children  | string |                       |          |             |
| className | string |                       |          |             |
| dataTest  | string | `'dhis2-uicore-help'` |          |             |
| error     | custom |                       |          |             |
| valid     | custom |                       |          |             |
| warning   | custom |                       |          |             |
