### Chip

#### Usage

**Note**: If possible, import the component from the main UI (`@dhis2/ui`) package.

```js
import { Chip } from '@dhis2-ui/chip'
```

#### Props

| Name      | Type     | Default               | Required | Description |
| --------- | -------- | --------------------- | -------- | ----------- |
| children  | any      |                       |          |             |
| className | string   |                       |          |             |
| dataTest  | string   | `'dhis2-uicore-chip'` |          |             |
| dense     | boolean  |                       |          |             |
| disabled  | boolean  |                       |          |             |
| dragging  | boolean  |                       |          |             |
| icon      | element  |                       |          |             |
| overflow  | boolean  |                       |          |             |
| selected  | boolean  |                       |          |             |
| onClick   | function |                       |          |             |
| onRemove  | function |                       |          |             |
